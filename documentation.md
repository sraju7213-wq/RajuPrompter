### Template System Rules

#### 1. **Template Structure**
Each template contains:
- **Title**: Descriptive name for the template
- **Description**: Brief explanation of use case
- **Tags**: Relevant keywords for categorization
- **Template String**: Parameterized prompt structure

#### 2. **Parameter Substitution**
Templates use placeholder variables:
```
{subject} - Main subject of the image
{lighting} - Lighting setup
{quality} - Quality parameters
{style} - Artistic style
{camera} - Camera/lens specifications
```

#### 3. **Quality Hierarchies**
Quality levels are automatically applied:
- **Professional**: Industry-standard specifications
- **High Quality**: Enhanced detail parameters
- **Masterpiece**: Premium quality descriptors
- **Ultra Detailed**: Maximum detail enhancement

### Library Management Rules
